# MyLife 管理员后台 — 使用指南

> 适用版本: v1.0  
> 日期: 2026-05-20

---

## 一、如何进入管理后台

1. 用管理员账号 `1258884229@qq.com` 正常登录 MyLife
2. 点击右上角 **头像按钮**，弹出下拉菜单
3. 在下拉菜单中，点击带盾牌图标的 **「管理后台」**
4. 进入 `/admin` 仪表盘页面

> 普通用户的下拉菜单只有「退出登录」，看不到「管理后台」入口。如果普通用户直接访问 `/admin` 网址，会被自动重定向回首页。

## 二、仪表盘

进入后台后默认显示的页面，顶部四张卡片一目了然：

| 卡片 | 含义 |
|------|------|
| **总用户数** | 平台上所有注册用户的总数 |
| **今日新增** | 今天注册的新用户数量 |
| **总写作数** | 平台上所有写作内容的总数 |
| **今日活跃** | 今天有过写作操作的用户数 |

往下滚动可以看到：

- **用户增长趋势**：近 30 天每天注册人数的柱状图，鼠标悬停可看到具体日期和数量
- **写作类型分布**：日记/随笔/小记/小说各自的占比，横向条形图
- **最近注册用户**：最新注册的 5 位用户邮箱和时间
- **最近写作**：平台上最新的 5 篇内容标题、类型和时间

> 仪表盘数据是只读的，纯粹用来快速了解平台概况。

## 三、用户管理

点击左侧导航「用户管理」进入。

### 3.1 查看用户列表

页面显示一个表格，包含所有注册用户：

| 列 | 内容 |
|----|------|
| 用户 | 首字母头像 + 邮箱地址 |
| 注册时间 | 用户注册日期 |
| 写作数 | 该用户总共写了多少篇 |
| 状态 | 绿色「正常」或红色「已禁用」 |
| 操作 | 「详情」按钮 |

底部有分页器，每页显示 20 个用户。

### 3.2 搜索用户

在顶部搜索框输入邮箱关键词，按回车搜索。比如输入 `qq.com` 列出所有 QQ 邮箱用户。

### 3.3 排序

搜索框旁边的下拉菜单可以切换排序方式：

- **最新注册**：最近注册的排在前面（默认）
- **最早注册**：最早注册的排在前面

### 3.4 查看用户详情

点击某个用户的「详情」按钮，右侧滑出详情面板，可以看到：

- 用户邮箱和 ID
- 注册时间和最后登录时间
- 总写作数量
- 当前状态（正常/已禁用）

面板底部有四个操作按钮：

### 3.5 禁用账号

1. 在用户详情面板，点击「禁用账号」
2. 用户将无法登录，状态变为「已禁用」

### 3.6 启用账号

1. 对已被禁用的用户，点击「启用账号」
2. 用户恢复正常登录权限

### 3.7 删除账号

1. 点击「删除账号及所有数据」
2. 弹出浏览器确认对话框，再次确认
3. 系统会删除该用户的：写作内容、小说项目、章节、角色、世界观设定、摘录、情绪记录、诗歌等所有数据
4. 最后删除用户账号本身

> 删除操作不可恢复，请谨慎使用。

### 3.8 查看用户内容

在详情面板点击「查看该用户内容」，会自动跳转到内容管理页面，并筛选出该用户的所有作品。

## 四、内容管理

点击左侧导航「内容管理」进入。

### 4.1 浏览内容

表格显示平台上所有用户的写作内容：

| 列 | 内容 |
|----|------|
| 类型 | emoji 图标 + 文字（日记/随笔/小记/小说章节） |
| 标题 | 文章标题，无标题时显示「无标题」 |
| 作者 | 作者的注册邮箱 |
| 字数 | 文章字数统计 |
| 时间 | 创建日期 |
| 操作 | 眼睛图标（查看）+ 垃圾桶图标（删除） |

### 4.2 筛选内容

顶部有一排类型标签，点击切换：

- **全部**：显示所有类型（默认）
- **日记** / **随笔** / **小记** / **小说**：只显示该类型

### 4.3 搜索内容

在搜索框输入关键词（按标题和正文内容同时搜索），按回车。比如搜索「春天」找出所有提到春天的文章。

### 4.4 排序

- **最新**：最近创建的排在前面（默认）
- **最早**：最早创建的排在前面
- **最长**：字数最多的排在前面

### 4.5 预览内容

点击某行的眼睛图标（第一颗按钮），右侧滑出预览面板：

- 显示文章类型标签、完整标题
- 作者、字数、创建时间
- 标签（如果有的话）
- 文章正文（最多显示前 2000 字，超出部分截断）
- 预览是**只读**的，不能编辑

### 4.6 删除内容

点击某行的垃圾桶图标，浏览器弹出确认框：

1. 点击「确定」：该内容被永久删除
2. 点击「取消」：不执行删除

> 删除操作不可恢复。

## 五、数据统计

点击左侧导航「数据统计」进入。

### 5.1 选择时间范围

右上角下拉菜单：

- **近 7 天**：只看最近一周的数据
- **近 30 天**：看最近一个月（默认）
- **近 90 天**：看最近三个月
- **全部**：看所有历史数据

### 5.2 图表说明

| 图表 | 看什么 |
|------|--------|
| **每日新增用户** | 蓝色柱状图，每天注册人数。柱越高代表那天新用户越多 |
| **每日写作量** | 绿色柱状图，每天平台上的写作数量。可以看出用户的写作活跃规律 |
| **写作类型分布** | 日记/随笔/小记/小说的数量占比，了解用户偏好 |
| **用户活跃时段** | 7天×24小时热力图。横轴是小时（0-23点），纵轴是周一到周日。颜色越深代表那个时段写作的人越多。可以用来发现用户的写作高峰期 |
| **Top 活跃用户** | 写作数量最多的 10 个用户排名。前三名会有高亮样式 |

### 5.3 导出报表

点击右上角「导出 CSV」按钮，浏览器会自动下载一个 CSV 文件，包含：

- 日期、新增用户、新增写作、总字数

这个文件可以用 Excel 或 WPS 打开，方便做进一步分析。

## 六、系统设置

点击左侧导航「系统设置」进入。

左侧有三个 Tab 标签页，点击切换。

### 6.1 AI 配置

控制平台的默认 AI 服务：

| 配置项 | 说明 |
|--------|------|
| **默认提供商** | Claude / OpenAI / DeepSeek 三个选项 |
| **默认模型** | 具体模型名称，如 Claude Sonnet 4.6 |

修改后点击下方的「保存配置」按钮生效。

### 6.2 功能开关

五个独立开关，点击右侧滑动按钮切换：

| 开关 | 打开时 | 关闭时 |
|------|--------|--------|
| **开放注册** | 新用户可以注册 | 注册功能关闭 |
| **公开分享** | 用户可以创建公开分享链接 | 分享功能关闭 |
| **AI 写作助手** | AI 续写/润色/改写可用 | AI 功能隐藏 |
| **音乐推荐** | 音乐推荐面板可用 | 音乐功能隐藏 |
| **维护模式** | 仅管理员可访问平台 | 所有用户正常访问 |

每个开关切换后是即时生效的（前端状态），但需要点击下方「保存设置」才能持久化到数据库。

### 6.3 平台公告

向所有用户发布通知：

1. 在输入框中输入公告文字
2. 点击「发布」按钮
3. 公告出现在下方列表中，显示内容和发布时间

已发布的公告可以点击垃圾桶图标删除。删除后不会在用户端显示。

## 七、个人账号管理

回到主站（点击左侧导航底部的「返回主站」），在首页左侧导航点击「设置」。

### 7.1 修改密码

1. 在「账号安全」区域找到「修改密码」卡片
2. 点击「修改」按钮
3. 输入新密码（至少 6 位）
4. 点击「确认修改」
5. 提示「密码修改成功」即完成

### 7.2 删除自己的账号

1. 在「账号安全」区域找到「删除账号」卡片（红色边框）
2. 点击「删除账号」按钮
3. 弹出确认对话框，提示所有数据将被永久删除
4. 点击「确认删除」

> 删除后会自动退出登录，跳转到登录页。

## 九、常见问题

**Q: 怎么添加其他管理员？**  
在 Supabase SQL Editor 执行：
```sql
UPDATE auth.users SET raw_app_meta_data =
  raw_app_meta_data || '{"is_admin": true}'::jsonb
WHERE email = '另一个邮箱@example.com';
```

**Q: 怎么取消某人的管理员权限？**  
```sql
UPDATE auth.users SET raw_app_meta_data =
  raw_app_meta_data - 'is_admin'
WHERE email = '某人@example.com';
```

**Q: 统计图表为什么是空白的？**  
数据需要时间积累。如果平台刚上线，用户量和写作量太少，部分图表（如热力图）可能因为数据不足显示为空白或颜色很浅。这是正常的，随着数据增长会逐渐丰富。

**Q: 为什么仪表盘加载很慢？**  
仪表盘需要遍历所有用户计算统计数据。如果用户量较大（数千以上），建议不要频繁刷新。后续可优化为定时任务缓存统计数据。
